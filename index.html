Supercharging Your Workflow: Coding with LLM Agents

Introduction - What are LLMs in Coding?

  • Large Language Models (LLMs) are AI systems trained on vast amounts of text and code.
  • In software development, they act as intelligent assistants.
  • They go beyond simple autocompletion to understand context, generate code, explain concepts, and even perform actions.
  • Think of them as a "pair programmer" that's available 24/7.

The Rise of the "Agentic" Coder

What does "Agentic" mean?

  • Proactive: It doesn't just wait for a command; it can anticipate needs.
  • Goal-Oriented: You provide a high-level goal (e.g., "create a login form"), and it takes steps to achieve it.
  • Tool-Using: It can interact with your file system, run commands, and read other files for context.
  • This is a shift from code suggestion to task automation.

In-IDE Agentic Tools

These LLMs live directly inside your code editor, providing seamless integration.

  • Cursor: An "AI-first" code editor built from the ground up for LLM integration.
  • GitHub Copilot: The most popular assistant, integrated into VS Code, JetBrains, and more.
  • Windsurf: A powerful agent that can understand the entire repository for deep contextual awareness.
  • Kilo Code: An emerging tool focused on intelligent, context-aware code generation.

Command-Line (CLI) Agents

For developers who live in the terminal, these agents bring LLM power to the command line.

  • Gemini CLI: An interactive chat experience in your terminal to ask questions, edit files, and run commands.
  • Codex: A pioneering model from OpenAI, known for its powerful natural language-to-code capabilities.
  • Claude Code: Anthropic's offering, providing strong coding assistance with a focus on safety and clear explanations.

When Are These Tools Most Useful?

LLM agents shine in specific, everyday coding scenarios:

  • Generating Boilerplate
  • Learning & Exploration
  • Debugging
  • Writing Unit Tests
  • Refactoring Code
  • Rapid Prototyping

The Ecosystem: Model Context Protocol (MCP)

MCP is an open-source standard for connecting AI applications to external systems.

  • The Core Problem It Solves: AI assistants can connect to data sources, tools and workflows
  • Enabling them to access key information and perform tasks:
    • Context Providers: Agents can access your Google Calendar and Notion, acting as a more personalized AI assistant.
    • Context Consumers: Claude Code can generate an entire web app using a Figma design.
  • The Impact: MCP results in more capable AI agents which can access your data and take actions on your behalf when necessary.
